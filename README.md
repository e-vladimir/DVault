# secure_storage
Аналог KeepassX и Keepass, которыми пользоваться не очень удобно из за странного UI.
Шифрование AES256 (base64)

Требования:
* python 3.x
* python3-PySide
* SimpleCrypto
* SQLite3

Если ругается на отсутствие драйвера SQLite:
* libqt4-sql-sqlite

Рабочее название проекта: DVault (DataVault)
